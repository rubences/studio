# **App Name**: AgentFlow

## Core Features:

- Content Rendering: Render HTML content in a structured, visually appealing format using Next.js.
- AI Agent Concepts: Provide access to curated content related to AI Agents, Google Cloud, and the ADK (Agent Development Kit).
- Agent Interaction Tool: Simulate interactions with AI agents, allowing users to input prompts and receive AI-generated responses that guide on Agent selection

## Style Guidelines:

- Primary color: Sky blue (#38bdf8) to evoke trust, innovation, and Google Cloud.
- Background color: Dark navy (#0f172a), creating a professional, tech-focused feel with high contrast for readability.
- Accent color: Electric indigo (#7c3aed) for interactive elements and important calls to action, providing visual interest and modernity.
- Headline font: 'Space Grotesk', sans-serif, for headlines and titles, reflecting a computerized, scientific aesthetic.
- Body font: 'Inter', sans-serif, for body text to ensure readability and a neutral, objective appearance.
- Use clean, line-based icons with the primary color (#38bdf8) to represent different AI agent types and functionalities, aiding navigation.
- Maintain a card-based layout to organize information. The site shall consist of a series of 'slides,' following the structure of the reference html.
